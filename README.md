# Task Manager Web App

A simple full-stack task management application where users can create, update, delete, and track tasks with due dates and priority levels. The project demonstrates CRUD operations and a RESTful API using Node.js, Express, and MongoDB.

## Features

- Add new tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- Set due dates and priority levels
- RESTful API backend

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript (can be upgraded to React later)

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB
